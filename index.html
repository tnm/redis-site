  • Version 2.2-alpha2 (preview of next 2.2 release) — September 13, 2010

Untar the tarball, cd into the Redis directory, and run make. Redis is now compiled.
Run it with ./redis-server.
If you are new to Redis, we have a nice online interactive tutorial.
